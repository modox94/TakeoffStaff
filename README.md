### Для запуска необходимо:
- Открыть `terminal` в каталоге `server`
  - Выполнить команду `npm i`
  - Выполнить команду `npm start`
- Открыть **ещё один** `terminal` в каталоге `client`
  - Выполнить команду `npm i`
  - Выполнить команду `npm start`

После этих действий по адресу http://localhost:3000/ запустится сайт. Для входа существует два тестовых пользователя:
- Name: `user1` Password: `pass1`
- Name: `user2` Password: `pass2`
